# Proyecto Pre-Entrega Final - Thiago

## 🚀 Docker Image
Puedes encontrar la imagen oficial aquí:
[(https://hub.docker.com/r/thiagocedres12/preentrega-thiago)]

### Cómo ejecutar con Docker:
1. `docker pull thiagocedres12/preentrega-thiago`
2. `docker run -p 8080:8080 thiagocedres12/preentrega-thiago`

## 📖 Documentación
Una vez ejecutado, visita:
`http://localhost:8080/api/docs`

## 🧪 Tests
Para correr los tests funcionales:
`npm test`
